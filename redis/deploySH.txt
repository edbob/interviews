Чтобы пользоватся db redis на сервере через ssh можно попробавать 
использоовать файл деплоя deploy.sh в него указываем соеденение :
ssh -f -N -L 6379:localhost:6379 -p 22 nameserver@xxx.xxx.x.XXX
и указываем в терминале команду start deploy.sh пишем пароль и вуаля, 
предворительно конечно сам redis и сервер должы быть запущены
доп пораметры для deploy.sh
  ssh root@178.62.251.8 '
  cd /root/10chan;
  git pull;
  npm install;
  forever stop index.js;
  forever start index.js
  '
